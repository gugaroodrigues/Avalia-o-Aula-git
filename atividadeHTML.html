Comandos mais utilizados no Git

git initInicializa um repositório Git
git clone [url]Clona um repositório remoto
git statusMostra o estado atual dos arquivos
git add .Adiciona todos os arquivos para o commit
git commit -m "mensagem"Registra alterações
git pushEnvia as alterações para o repositório remoto
git pullAtualiza o repositório local
git logExibe o histórico de commits

Resumo dos aprendizados:
Até o momento no curso tivemos alguns aprendizados interessantes, atualmente estamos tendo aulas de git, introdução a dispositivos movéis e metodologias ágeis. De maneira geral posso resumir alguns aprendizados tais como os de git aonde aprimorei meus conhecimentos, sobre versionamento, branch, e como trabalhar em equipe usando o git. Em metodologias ágeis posso afirmar que o mais impactante para mim foi descobrir como utilizar o "Scrum" modelo este que seria de grande ajuda anos atras quando finalizei o meu projeto do entra21 Em introdução a dispositivos móveis, tivemos alguns problemas com o ambiente de trabalho dentro da instituição de ensino mas posso concluir dizendo que um dos aprendizadosmais importantes que obtive foi aprender a criar um senso mais critico como desenvolvedor, conseguindo analisar e criar um relatorio sobre certos aplicativos, também estimulando a minha curiosidade e o autodidatismo, aonde fizemos uma tela de login, aonde fui atras e a incorporei com outra atividade de metodologias ágeis. Tenho buscado em contribuir de algumas maneiras, sempre tentando participar, seja questionando ou interagindo com com os colegas em classe, além de compartilhar o pouco que sei da maneira que posso, convidando-os colegas a entrarem no discord para discutir sobre algumas atividades e ajudar uns aos autros.
